API reference for Angular Material button

import { MdButtonModule } from '@angular/material';

Directives

MdButton

A Material Design button.

Selectors:

Attribute Description
button[md-button] / a[md-button]

Rectangular button w/ no elevation.
button[md-raised-button] / a[md-raised-button]

Rectangular button w/ elevation
button[md-icon-button] / a[md-icon-button]

Circular button with a transparent background, meant to contain an icon
button[md-fab] / a[md-fab]

Circular button w/ elevation, defaults to theme's accent color
button[md-mini-fab] / a[md-mini-fab]

Same as md-fab but smaller
