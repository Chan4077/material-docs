API Reference for Angular Material Datepicker

import { MdDatepickerModule, MdNativeDateModule } from '@angular/material';

Services

MdDatepickerIntl

Datepicker data that requires internationalization.

Directives

MdDatepicker

Component responsible for managing the datepicker popup/dialog.

Selector: mat-datepicker MdDatepickerInput

Directive used to connect an input to a MdDatepicker.

Selector: input[matDatepicker] MdDatepickerToggle

The toggle for the datepicker.

Selector: button[matDatepickerToggle]

Note: In master, the selector has changed to mat-datepicker-toggle.
