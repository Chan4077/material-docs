API reference for Angular Material button

import { MatButtonModule } from '@angular/material'; import { MatButtonModule } from '@angular/material/button';

Directives

MatButton

A Material Design button.

Selectors:

Attribute Description
button[mat-button] / a[mat-button]

Rectangular button w/ no elevation.
button[mat-raised-button] / a[mat-raised-button]

Rectangular button w/ elevation
button[mat-icon-button] / a[mat-icon-button]

Circular button with a transparent background, meant to contain an icon
button[mat-fab] / a[mat-fab]

Circular button w/ elevation, defaults to theme's accent color
button[mat-mini-fab] / a[mat-mini-fab]

Same as mat-fab but smaller
