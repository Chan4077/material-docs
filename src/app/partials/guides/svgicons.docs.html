Getting Started with SVG Icons

  1. Add MdIconModule in your app's app.module.ts:
    2. import { MdIconModule } from '@angular/material'; // Your other app's modules // ... @NgModule([ imports: [ MdIconModule ] ]); export class AppModule { }

How do I load SVG iconsets in my app?

Follow these steps:

  1. Download an svg icon or svg iconset from another source (such as MaterialDesignIcons).
  2. Place it somewhere in your assets folder.
  3. In your app's AppModule, add the following code:
    4. import { MdIconRegistry } from '@angular/material'; import { DomSanitizer } from '@angular/platform-browser'; export class AppModule { constructor(private mdIconRegistry: MdIconRegistry, private domSanitizer: DomSanitizer){ // Replace with the path to your icons mdIconRegistry.addSvgIconSet(domSanitizer.bypassSecurityResourceUrl('assets/icons.svg')); } }
  4. To use your icon, add the following line:
    5. <mat-icon svgIcon="icon"></mat-icon>

How do I load a single SVG icon then?

It's the same as the above section How do I load SVG icon sets in my app? except the third step:

import { MdIconRegistry } from '@angular/material'; import { DomSanitizer } from '@angular/platform-browser'; export class AppModule { constructor(private mdIconRegistry: MdIconRegistry, private domSanitizer: DomSanitizer){ // Replace with the path to your icons mdIconRegistry.addSvgIcon(domSanitizer.bypassSecurityResourceUrl('assets/icon.svg')); } }
